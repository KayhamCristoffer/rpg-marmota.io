# CHANGELOG

## v1.0.0 — 2026-03-02

### Adicionado
- Login via Discord OAuth2 (Passport.js + express-session)
- Sistema de quests: Diária, Semanal, Mensal, Evento
- Upload de comprovante com Multer (5MB, imagens)
- Sistema de XP, Níveis e Badges
- Ranking por período: Total / Diário / Semanal / Mensal
- Reset automático via CRON (node-cron)
- Painel Admin completo (CRUD quests, revisão de prints, gestão de usuários)
- Tema RPG dark com bordas douradas
- Interface responsiva (mobile-first)
- 4 models MongoDB: User, Quest, UserQuest, Ranking
